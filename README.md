# 🖥️ Terminal Style Portfolio
A sleek, Linux-inspired terminal portfolio website featuring a functional login system, interactive command-line interface, and real-time system monitoring vibes.
This project allows users to explore my background, projects, and skills through a functional Command Line Interface (CLI) built for the web.

To view my portfolio => https://waiyan-n0.github.io/terminal-portfolio/

![License](https://img.shields.io/badge/license-MIT-green)
![Status](https://img.shields.io/badge/status-active-success)

## ✨ Features

*   **Interactive CLI:** A fully functional terminal experience directly in your browser.
*   **Intelligent Autocomplete:** Uses the **Levenshtein Distance Algorithm** to provide fuzzy matching and command suggestions even if you make a typo.
*   **Custom Commands:** Includes built-in commands like `help`, `about`, `projects`, `contact`, and `clear`.
*   **Custom Login:** Includes Login Page for viewer and used LocalStorage for user. `whoami` command is for login user
*   **Responsive UI:** Optimized for both desktop and mobile screens with a high-contrast "Ubuntu Linux" inspired aesthetic.

## 🚀 Tech Stack

*   **Frontend** HTML5 CSS3 JavaScript
*   **Styling**: Custom CSS with Flexbox for responsive alignment.
*   **Algorithms:** Levenshtein Distance for "Fuzzy Search" and command prediction.
*   **Deployment:** GitHub Pages
